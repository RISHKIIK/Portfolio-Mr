Project SCREWED & BREWED

AdobeXD Design for coffe Application

Designing an application for displaying coffee available at a store.


View the Live Design :


Designing UI interface,mobile design using AdobeXD for a coffee shop.Basic scenario description stated below:

Functionality: The first page displays options for viewing and selecting coffee options according to your choice.

i) App allows to choosing Coffee and its quantity and place it in cart.

ii) There is option to view cart by dragging the below arrow upwards.

iii) The cart view shows all your choosen coffees and allows you to choose sugar milk of your choice by switching between tabs.UI is made user friendly.

iv) On clicking on continue button.The page shifts to payment options page.There are various methods depiced using which you can make the payment.

TECH STACK

This project is a UI Design build using AdobeXD.

Modern design is built using the concepts like motherboard formation and Prototype building.
