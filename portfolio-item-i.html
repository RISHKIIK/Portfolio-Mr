Project GOOGLE CHROME CLONE

An interactive live Reactjs Application

Creating a clone app similar to google chrome new tab.

Creating a clone app with same UI as google chrome tab.

Functionality: Need to able to type something in search-box and when user hits enter button it should be added as recently viewed tabs that shows search result below the search box. Recent 10 search results should be shown at max

This project was bootstrapped with Create React App. Tech stack used is ReactJs for building the UI, Mainly ReactJs16 webhooks are taken in account.

Modern react concepts like Higher order components, React context API Hooks , usage of custom hooks and reusable compoents are taken into account.Material-Ui for icons and scss is used for Application creation.
